# KLINIK DR SITI DAN RAKAN-RAKAN

Laman web rasmi untuk **KLINIK DR SITI DAN RAKAN-RAKAN**, dibangunkan sebagai aplikasi web moden berprestasi tinggi menggunakan teknologi terkini bagi memastikan pengalaman pengguna yang pantas, responsif, dan selamat.

---

## Maklumat Klinik

**Nama:** KLINIK DR SITI DAN RAKAN-RAKAN
**Jenis:** Klinik Kesihatan & Rawatan Umum
**Platform:** Web Application (Single Page Application)

---

## Tentang Projek

Repositori ini mengandungi kod sumber penuh bagi laman web klinik yang direka untuk:

* Menyampaikan maklumat klinik secara profesional
* Meningkatkan kepercayaan pesakit melalui paparan testimoni dan profil doktor
* Memudahkan komunikasi melalui integrasi WhatsApp
* Memberikan pengalaman pengguna yang lancar pada semua peranti

---

## Teknologi Digunakan

* React 19
* Vite
* React Router
* Tailwind CSS
* Framer Motion
* GSAP
* Swiper
* Lucide React

---

## Performance & Optimization

Website ini dioptimumkan berdasarkan analisis Google PageSpeed Insights:

* Performance: 99 (Mobile)
* Accessibility: 100
* Best Practices: 100
* SEO: 100

### Core Web Vitals

* Largest Contentful Paint (LCP) < 2.5 saat
* Cumulative Layout Shift (CLS) hampir 0
* Interaction to Next Paint (INP) dioptimumkan

### Strategi Optimasi

* Code splitting menggunakan React lazy loading
* Asset bundling dan compression melalui Vite
* Penggunaan imej format moden dan lazy loading
* Animasi dioptimumkan menggunakan GPU acceleration (GSAP)
* Struktur komponen modular untuk render lebih efisien

---

## Security Implementation

Website ini dilengkapi dengan konfigurasi keselamatan moden bagi melindungi pengguna dan sistem.

### Security Headers Digunakan

* Content Security Policy (CSP)
* X-Frame-Options
* X-Content-Type-Options
* Referrer-Policy
* Strict-Transport-Security (HSTS)

### Perlindungan Yang Dicapai

* Mengurangkan risiko Cross-Site Scripting (XSS)
* Mengelakkan serangan clickjacking
* Menghalang MIME sniffing
* Melindungi penghantaran data melalui HTTPS

---

## System Architecture

Sistem ini direka dengan pendekatan modular dan scalable.

* React digunakan untuk membina struktur komponen yang fleksibel
* Vite digunakan untuk build system yang pantas dan efisien
* Tailwind CSS memastikan konsistensi rekaan
* GSAP dan Framer Motion digunakan untuk animasi berprestasi tinggi

Struktur ini memudahkan penambahan ciri baharu tanpa menjejaskan prestasi sistem sedia ada.
